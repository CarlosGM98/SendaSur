<h2 align="center">SendaSur - Aplicación Web de Rutas de Senderismo</h2>

<p align="center">
SendaSur es una aplicación web enfocada en el mundo del <strong>senderismo</strong>. 
Tras registrarse e iniciar sesión, los usuarios pueden crear, modificar y eliminar rutas.
</p>

<p align="center">
Cada ruta incluye información detallada como:
</p>

<p align="center">
✔ Nombre de la ruta <br>
✔ Descripción <br>
✔ Localización (integración con Google Maps) <br>
✔ Dificultad (según desnivel y complejidad) <br>
✔ Imágenes <br>
✔ Valoración (1 a 5 ⭐) <br>
✔ Comentarios de usuarios
</p>
